## Salesforce geral

Aprenda sobre novos recursos e aprimoramentos que afetam sua experiência geral do Salesforce.

* **Melhorias gerais**

* **Gire seus certificados com mais frequência** — _Disponível para admins/devs_

* **Validar informações de domínio com mais frequência** — _Disponível para admins/devs_

* **Atualizar seus certificados mTLS** — _Disponível para admins/devs_

* **Preparar-se para IPv6** — _Disponível para admins/devs_

* **Localize e implemente soluções facilmente com marcas de categoria e seleção guiada** — _Requer configuração_

* **Chatter é desativado por padrão em novas organizações** — _Disponível para admins/devs_

* **Habilitar aprimoramentos de acessibilidade para cabeçalhos de página e janelas modais quando o zoom for maior que 200% (atualização de versão)** — _Requer configuração_

* **Habilitar aprimoramentos de acessibilidade para seletores de data, popovers, barras de utilitários inferior, cabeçalhos de registro (atualização de versão)** — _Requer configuração_

* **Habilitar aprimoramentos de acessibilidade para cartões, contêineres encaixados, listas de menus e painéis (atualização de versão)** — _Requer configuração_

* **Habilite os aprimoramentos de acessibilidade para listas de tarefas e listas duplas do Lightning quando o zoom for maior que 200% (atualização de versão)** — _Requer configuração_

* **Arquivar aplicativo**

* **Monitore o progresso do arquivo com novos status de atividade**

* **Gerenciar dados com mais eficiência com as novas configurações do aplicativo Arquivar**

* **Renomear recursos do aplicativo de arquivo para se alinhar aos padrões do Salesforce**

* **Prepare-se para a descontinuação de seu próprio pacote gerenciado de arquivo e arquivo legado**

* **Salesforce Foundations**

* **Priorizar seus principais leads com pontuação de pessoas em fundamentos** — _Disponível para admins/devs_

* **Transformar o tráfego da Web em percepções acionáveis com rastreamento da Web** — _Disponível para admins/devs_

* **Pipelines de dados do Salesforce**

* **Aumente o desempenho e a confiabilidade com o Inspetor de receita de pipelines de dados do Salesforce** — _Disponível para usuários_

* **Exportar agendas de receita para seu calendário externo para fácil rastreamento** — _Disponível para usuários_

* **Simplifique o backup de receita com upload e download diretos no Gerenciador de dados** — _Disponível para usuários_

* **Melhore o desempenho da receita de dados de instantâneo com ações de inserção e inserção e exclusão otimizadas (disponível ao público em geral)** — _Disponível para usuários_

* **Exportar dados de pipelines de dados do Salesforce para o Azure Data Lake (disponível ao público em geral)** — _Disponível para usuários_

* **Salesforce Scheduler**

* **Mostrar a posição específica da categoria em cartões do participante da lista de espera** — _Requer configuração_

* **Ajude os usuários a resolver erros de compromisso mais rapidamente com mensagens de validação personalizadas** — _Requer configuração_

* **Mantenha componentes de lobby personalizados sincronizados com seleções de território** — _Requer configuração_

* **Salesforce Meu centro de Trust**

* **Mantenha-se atualizado com o Salesforce My Trust Center (disponível ao público em geral)** — _Disponível para usuários_
