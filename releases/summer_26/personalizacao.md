## Personalização

Permissões de modo de exibição de lista mais granulares permitem que os usuários compartilhem seus modos de exibição de lista sem permitir que editem todos os modos de exibição de lista da sua organização. Ajude seus usuários a se manterem sincronizados graças ao suporte para mais 27 fusos horários canônicos de acordo com os padrões IANA. Obtenha autenticação mais robusta e segura com o suporte do adaptador organizacional cruzado do Salesforce Connect para credenciais nomeadas.

* **Recursos de personalização lançados por mês**

* **AgentExchange**

* **Não há suporte para novas páginas de login do Trialforce com marca no Cloudforce.com** — _Disponível para admins/devs_

* **Serviços externos**

* **Crie integrações mais robustas com o suporte de serviços externos para enumerações** — _Disponível para admins/devs_

* **Faça mais com as melhorias de arquivo binário de Serviços externos** — _Disponível para admins/devs_

* **Globalização**

* **Oferecer suporte a mais fusos horários** — _Disponível para admins/devs_

* **Expanda traduções do idioma do usuário final para catalão e basco (beta)** — _Disponível para admins/devs_

* **Revise traduções atualizadas de rótulos** — _Disponível para admins/devs_

* **Habilitar os formatos de localidade de ICU (atualização de versão)** — _Disponível para admins/devs_

### Permissões

* **Revise o acesso ao campo entre perfis, conjuntos de permissões e grupos de conjuntos de permissões** — _Disponível para admins/devs_

* **Rastrear dependências de permissão mais facilmente** — _Disponível para admins/devs_

* **Habilitar filtragem de perfil (atualização de versão)** — _Requer configuração_

* **Monitore alterações administrativas em perfis com políticas de segurança da transação** — _Disponível para admins/devs_

* **Salesforce Connect**

* **Crie integrações mais seguras com suporte de adaptador entre organizações para credenciais nomeadas** — _Disponível para admins/devs_

* **Configurar com o Agentforce**

* **O agente legado para configuração não pode mais ser habilitado** — _Requer configuração_

* **Compartilhamento**

* **Escolher se deseja conceder acesso usando hierarquias de papéis para filas** — _Disponível para admins/devs_

* **Atualize padrões organizacionais de modo mais rápido e confiável** — _Disponível para admins/devs_

* **Ampliar o acesso para usuários de licença de funcionário unificado** — _Disponível para admins/devs_

* **Atualize o Apex code e os fluxos para alterar o comportamento de recálculo de compartilhamento (atualização de versão)** — _Requer configuração_

* **Configuração geral**

* **Faça mais com campos personalizados em entidades padrão** — _Disponível para admins/devs_

* **Gerenciar compartilhamento e edição de modo de exibição de lista com permissões granulares** — _Disponível para admins/devs_
