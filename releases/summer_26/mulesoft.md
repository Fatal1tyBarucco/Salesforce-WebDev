## MuleSoft

* **Catálogo de API para Salesforce**

* **Mapear seus agentes para ferramentas do servidor MCP** — _Disponível para admins/devs_

* **Trazer servidores MCP do MuleSoft para o catálogo de API (disponível ao público em geral)** — _Disponível para admins/devs_

* **Descubra servidores MCP registrados manualmente no catálogo de API (disponível ao público em geral)** — _Disponível para admins/devs_

* **Visualizar APIs de consulta nomeadas no catálogo de API e ativar ações (disponível ao público em geral)** — _Disponível para admins/devs_

* **Inteligência de integração do MuleSoft** — _Disponível para admins/devs_
