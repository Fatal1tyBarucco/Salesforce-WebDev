## Partner Cloud

Comece a gerenciar o ciclo de vida completo do parceiro em um só lugar com ferramentas integradas diretamente no Salesforce. Use o Partner Cloud para recrutar, integrar, habilitar e co-venda com parceiros enquanto dá a eles acesso seguro aos dados e fluxos de trabalho de que precisam. Reúna os principais recursos, como registro de negócio, incentivos, treinamento e rastreamento de desempenho, para que sua equipe possa escalar vendas indiretas e melhorar os relacionamentos com parceiros. O Partner Cloud ajuda você a fazer mais com os parceiros que você já tem, com mais rapidez e com menos trabalho manual.

* **Criar e rastrear planos de negócios conjuntos com seus parceiros** — _Disponível para usuários_

* **Incluir códigos de indicação em registros de negócio** — _Disponível para usuários_

* **Monitore o desempenho do agente de parceiro com o Agent Analytics** — _Disponível para usuários_
