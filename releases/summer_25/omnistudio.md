## OmniStudio

O OmniStudio oferece designers padrão e modos de exibição de lista mais rápidos para todos os componentes. O designer padrão para OmniScripts também oferece suporte ao Newport Design System para designs personalizados mais fáceis. Expandindo o suporte para vários canais, os OmniScripts agora podem ser implementados em sites do Lightning Web Runtime (LWR) para Experience Cloud. Os Mapeadores de dados do OmniStudio podem carregar dados de origens externas usando um objeto de interface personalizado como entrada. A solução de problemas de etapas com falha em Procedimentos de integração ficou mais fácil com o registro detalhado. Para facilitar a migração do tempo de execução de pacote gerenciado para o tempo de execução padrão, o OmniStudio converte automaticamente funções personalizadas em procedimentos de integração e mapeadores de dados. Com os novos métodos, agora você pode usar as APIs REST do Connect para chamar Mapeadores de dados ou Procedimentos de integração em tempo de execução padrão.
