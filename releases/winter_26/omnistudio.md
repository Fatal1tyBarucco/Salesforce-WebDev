## OmniStudio

O OmniStudio oferece FlexCards e OmniScripts em sites do Lightning Web Runtime (LWR) do Experience Cloud como um recurso beta. Use a nova página Número automático global Omni para criar sistemas de numeração exclusivos para necessidades de negócios específicas. Os componentes Flexcard, Omniscript e OmniStudio PubSub Lightning agora estão disponíveis diretamente na Biblioteca de componentes do Salesforce Lightning e podem ser adicionados aos seus projetos personalizados do componente da Web Lightning (LWC). Aumente o desempenho de tempo de execução de seus componentes usando a configuração Desempenho de tempo de execução aprimorado.
