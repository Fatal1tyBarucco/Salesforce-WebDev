## Experience Cloud

Aproveite os recursos mais recentes do Lightning Web Runtime atualizando seus sites do LWR para o LWR aprimorado. Novos recursos ajudam a garantir a disponibilidade consistente de imagens em fluxos de tela em sites do Aura e do LWR e oferecem uma visualização de como os fluxos aparecem com a identidade visual do seu site. Para garantir que todos os visitantes tenham acesso contínuo ao seu site, atualize as referências aos URLs do site que terminam no nome antigo do host Force.com. Use o Desenvolvedor local para criar componentes da Web Lightning mais rapidamente para seu site do LWR. Novos recursos para aplicativos Mobile Publisher melhoram a experiência do usuário e aumentam a segurança do aplicativo.

* **Atualizar para sites do LWR aprimorados (atualização de versão)** — _Requer configuração_

* **Aproveite os recursos mais recentes do Salesforce Flow para Experience Cloud** — _Disponível para admins/devs_

* **Crie sites e aplicativos eficientes com componentes predefinidos do Avonni** — _Requer configuração_

* **Atualizar referências aos URLs do seu site Force.com legado** — _Disponível para admins/devs_

* **Informar os usuários quando a sessão está prestes a terminar** — _Disponível para admins/devs_

* **Desenvolva componentes da Web Lightning mais rapidamente em uma visualização em tempo real do seu site do LWR (beta)** — _Requer configuração_

* **Mudar para um certificado de domínio único para sua rede de distribuição de conteúdo do Salesforce (atualização de versão)** — _Disponível para admins/devs_

* **Descubra os recursos mais recentes dos aplicativos Mobile Publisher** — _Requer configuração_
