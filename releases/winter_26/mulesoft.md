## MuleSoft

Use o pacote de produtos MuleSoft Anypoint Platform para conectar e integrar aplicativos, sistemas e dados em sua empresa. Simplifique operações criando e automatizando processos com cliques, em vez de código. Você pode projetar, desenvolver, administrar e compartilhar APIs e aplicativos de integração e hospedá-los na nuvem ou no local.

* **Catálogo da API do MuleSoft para Salesforce**

* **Autenticar conexões de API usando Básico, JWT e OAuth 2.0** — _Disponível para admins/devs_

* **Empacotar entidades do catálogo de API usando pacotes gerenciados de primeira ou segunda geração**

* **Selecionar conexões de API por ação no Fluxo** — _Disponível para admins/devs_
