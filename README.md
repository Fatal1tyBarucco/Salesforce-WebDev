# Salesforce-WebDev

Salesforce Release Intelligence Platform.

## Objetivos

- Automatizar captura de Salesforce Release Notes
- Estruturar Knowledge-as-Code
- Versionar mudanças arquiteturais Salesforce
- Centralizar documentação técnica navegável

## Features

- Release Notes Scraping
- Topic Classification
- Markdown Generation
- GitHub Actions Automation
- Dynamic README Builder
- GitHub Pages Foundation
- Structured Logging
- Retry HTTP Client

## Estrutura

```text
/releases/{release}/{topic}.md
```
