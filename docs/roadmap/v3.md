# Roadmap V3

## Objectives

- Production hardening and reliability
- Historical analytics and trend visualization
- API layer for programmatic access
- Stakeholder notifications
- Interactive exploration experience
- Review-based release workflow

## Planned Features

### Production Hardening ✅
- ✅ Rate limiting for Salesforce API calls — token-bucket at 2 req/s
- ✅ Circuit breaker pattern — trips after 3 failures, 60s cooldown
- ✅ Graceful degradation — serve cached data when live fetch fails
- ✅ Structured logging with correlation IDs — JSON + text formatters
- ✅ Health check endpoint — /health with full status, /ready probe
- ✅ Prometheus metrics export — /metrics with counters and gauges

### Historical Analytics ✅
- ✅ Trend charts — feature count growth per category over releases
- ✅ Category velocity — which categories grow fastest
- ✅ Confidence trend — parse quality improving or degrading over time
- ✅ Release cadence tracker — days between releases
- ✅ Export analytics dashboard as static HTML

### API Layer ✅
- ✅ REST API (stdlib) to query release data
- ✅ GET /releases — list all releases with metadata
- ✅ GET /releases/{slug} — full release details
- ✅ GET /releases/{slug}/categories/{name} — category features
- ✅ GET /diff/{current}/{previous} — comparison between releases
- GraphQL endpoint for flexible queries
- OpenAPI/Swagger documentation auto-generated

### Stakeholder Notifications ✅
- ✅ Email digest after each release (SMTP)
- ✅ Slack webhook integration
- ✅ Discord webhook integration
- ✅ Configurable notification profiles per release category
- ✅ Unsubscribe management

### Interactive Web Dashboard ✅
- ✅ Static HTML dashboard with inline JavaScript
- ✅ Feature search across all releases
- ✅ Category filter and drill-down
- ✅ Side-by-side release comparison UI
- ✅ Confidence heatmap visualization
- ✅ Export filtered results as CSV/JSON

### Review-Based Workflow ✅
- ✅ Pipeline creates PR instead of direct push
- ✅ Automated PR with diff preview
- ✅ Required review before merge
- ✅ Auto-merge after approval for minor changes
- ✅ Label-based triage (new features, regressions, config-required)

### Salesforce Integration
- Link features to Trailhead modules
- Auto-detect new Trailhead content related to release features
- Cross-reference feature availability with org limits
- Sandbox deployment readiness checker
