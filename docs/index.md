# Salesforce Release Intelligence

Portal técnico automatizado para versionamento e classificação de Salesforce Release Notes.

## Objetivos

- Automatizar captura de release notes
- Estruturar Knowledge-as-Code
- Centralizar mudanças arquiteturais Salesforce
- Versionar conteúdo técnico

## Features

- GitHub Actions
- Release Discovery
- Semantic Classification
- Markdown Artifact Generation
- GitHub Pages Ready
